# МедДневник

Персональный веб-дневник для ведения медицинских анализов крови и мочи. Позволяет хранить историю лабораторных исследований, отслеживать динамику показателей в виде графиков и автоматически сравнивать результаты с референсными нормами.

---

## Возможности

### Журнал анализов
- Добавление, редактирование и удаление записей об анализах
- Поддержка 34 лабораторных показателей, разбитых по разделам:
  - **Биохимия крови** — глюкоза, триглицериды, холестерин (общий, ЛПВП, ЛПНП, ЛПОНП), АЛТ, АСТ, креатинин, коэффициент де Ритиса, соотношение ЛПНП/ЛПВП
  - **Клинический анализ (ОАК)** — гемоглобин, эритроциты, гематокрит, СОЭ, тромбоциты, лейкоциты, нейтрофилы, лимфоциты, моноциты, эозинофилы и другие индексы (MCV, MCH, MCHC, RDW, MPV, PDW)
  - **Общий анализ мочи** — цвет, прозрачность, pH, белок, глюкоза, кетоны, билирубин, лейкоциты, эритроциты и другие параметры
  - **Серология** — Anti-HCV, HBsAg, Anti-HBs
- Отметка о диете на момент сдачи анализа (натощак / обычное питание)
- Просмотр полного результата одного анализа с цветовой подсветкой отклонений от нормы

### Импорт из PDF
- Автоматическое распознавание показателей из PDF-файлов лабораторий
- Умный парсер сопоставляет строки PDF с полями модели по ключевым словам (на русском и английском)
- При добавлении анализа можно загрузить несколько PDF — данные объединяются в одну запись

### Дашборд с графиками
- Интерактивные графики динамики любых выбранных показателей (Chart.js)
- Возможность выбрать несколько показателей для сравнения на одном графике
- Карточка «Анализ динамики»: последние значения выбранных параметров, их статус по норме, наибольшие изменения между последними двумя анализами

### Нормы и интерпретация
- Для каждого показателя задан референсный диапазон
- При отклонении выводится описание возможных причин и рекомендации
- Автоматически рассчитываемые показатели: коэффициент де Ритиса (АСТ/АЛТ) и индекс атерогенности (ЛПНП/ЛПВП)

### Учётные записи
- Регистрация и авторизация пользователей (Flask-Login)
- Каждый пользователь видит только свои данные
- Редактирование профиля: имя, email, смена пароля

### Панель администратора
- Управление пользователями (создание, редактирование, удаление, назначение роли администратора)
- Просмотр базы данных прямо в браузере
- Экспорт всей базы в JSON и импорт из JSON (резервное копирование)

## Установка на сервер (Debian/Ubuntu)

```bash
curl -sSL https://raw.githubusercontent.com/shmvme-cmd/Blood_readings_web/master/install.sh | sudo bash
```

Скрипт автоматически:
1. Устанавливает системные зависимости (curl, wget)
2. Устанавливает пакетный менеджер [uv](https://github.com/astral-sh/uv)
3. Скачивает проект с GitHub
4. Создаёт Python-окружение и устанавливает зависимости
5. Генерирует `SECRET_KEY` в файл `.env`
6. Скачивает статические ресурсы (Bootstrap, Chart.js, Flatpickr) в `static/`
7. Создаёт и запускает systemd-сервис `med-dnevnik`

После установки приложение доступно по адресу: `http://<IP>:5001`

Управление сервисом:
```bash
systemctl status med-dnevnik
systemctl restart med-dnevnik
journalctl -u med-dnevnik -f
```

## Локальный запуск (для разработки)

```bash
# 1. Клонировать репозиторий
git clone https://github.com/shmvme-cmd/Blood_readings_web.git
cd Blood_readings_web

# 2. Создать окружение и установить зависимости
uv venv
uv pip install -e .

# 3. Задать секретный ключ
echo "SECRET_KEY=your-secret-key" > .env

# 4. Запустить
python run.py
```

Приложение откроется на `http://127.0.0.1:5001`
